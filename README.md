## 📖 Sobre o projeto

Este projeto foi criado para documentar conhecimentos básicos de **suporte técnico e redes de computadores**.

O site apresenta comandos de diagnóstico de rede e orientações básicas utilizadas em atividades de suporte técnico.

---

## 📚 O que você encontrará no site

Ao acessar o projeto, você poderá clicar nas seguintes opções:

### 🔹 Configurar IP
Explicação do comando **ipconfig**, utilizado para visualizar informações de rede como endereço IP, máscara de sub-rede e gateway padrão.

### 🔹 Testar Conexão
Explicação do comando **ping**, utilizado para verificar se há conexão com a internet.

### 🔹 Ver rota da conexão
Explicação do comando **tracert**, que mostra o caminho que os dados percorrem até chegar ao destino.

### 🔹 Formatar PC
Passos básicos para formatação de um computador e instalação do sistema operacional.

### 🔹 Instalar Linux
Guia básico para instalação de uma distribuição Linux utilizando um pendrive bootável.
