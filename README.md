![Objetivo](https://img.shields.io/badge/OBJETIVO-0A1AFF?style=for-the-badge)

Criar um guia simples para estudo de **suporte técnico e redes de computadores**, mostrando comandos de diagnóstico e procedimentos básicos utilizados na área de TI.

---

![Conteudo](https://img.shields.io/badge/CONTEÚDO-0A1AFF?style=for-the-badge)

Ao acessar o site você encontrará:

- 🔹 Configurar IP  
- 🔹 Testar Conexão  
- 🔹 Ver rota da conexão  
- 🔹 Formatar PC  
- 🔹 Instalar Linux  

Cada opção apresenta uma explicação básica sobre o procedimento ou comando utilizado.
