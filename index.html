Guia Básico de Suporte e Redes

Projeto de estudo de suporte técnico e redes

Acesse as opções abaixo para aprender comandos e procedimentos básicos.

Configurar / Verificar IP

Abra o Prompt de Comando e digite:

ipconfig

Esse comando mostra:

  • Endereço IP
  • Máscara de sub-rede
  • Gateway padrão

Testar Conexão

Para testar se a internet está funcionando digite:

ping 8.8.8.8

Se aparecer resposta significa que a conexão está funcionando.

Ver rota da conexão

Digite o comando:

tracert google.com

Esse comando mostra por quais servidores sua conexão passa até chegar ao destino.

Formatar um computador

  2. Entrar na BIOS

  4. Instalar o sistema operacional
